IDM+User Agent Switcher下载文件

发表于 2016-11-05

一些文件只能用客户端下载或者服务器限制了下载速度，这时可以用IDM下载软件和Firefox插件User Agent Switcher配合使用，下载速度可大幅度增加甚至做到满速下载

准备工作

  1. IDM(Internet Download manager)可下载原版破解或直接下载破解版，具体方法请自行百度。
  2. Firefox浏览器
  3. User Agent Switcher插件(2011年的旧版本，不要选择新版本)

第一步
首先安装Firefox浏览器，并安装User Agent Swticher插件

按Alt键-工具，在Default User Agent里面选择iPhone 3.0

现在打开网页的效果与使用手机打开是一样的。

这时完成了第一个步骤，接下来到第二个步骤。

第二步

安装IDM

最后就可以下载我们想要的文件了
